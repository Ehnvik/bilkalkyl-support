## Om appen

**Bilkalkyl** är en enkel och tydlig bilkalkylator för privatpersoner och företag.

Appen hjälper dig att räkna ut total månadskostnad för billån eller leasing genom att ange:

- Bilens pris
- Restskuld
- Ränta
- Avgifter

För företag visas även kostnad exklusive moms.

Version: 1.0  
Utvecklare: Gustav Ehnvik

---

## Privacy Policy

Bilkalkyl samlar inte in, lagrar eller delar någon personlig information.

Appen:

- Kräver inget konto
- Samlar inte in personuppgifter
- Använder ingen tredjepartsanalys
- Spårar inte användare

All beräkning sker lokalt på användarens enhet.

Om detta ändras i framtida versioner kommer denna sida att uppdateras.
